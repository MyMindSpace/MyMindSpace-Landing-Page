# MyMindSpace - Your Mind, Unfolded.

![MyMindSpace Banner](public/banner.png)

**MyMindSpace** is a premium, AI-powered mental wellness companion designed to help you unfold your mind. This project features a cinematic "scrollytelling" landing page where a journal visually evolves as you scroll, creating a serene and immersive user experience.

## ‚ú® Features

- **Cinematic Scrollytelling**: A highly polished scroll-linked animation sequence that tells a story as the user explores the page.
- **Buttery Smooth Scrolling**: Integrated with [Lenis](https://github.com/darkroomengineering/lenis) for a luxurious scroll feel.
- **Premium Aesthetics**: Minimalist, high-end design with grain overlays, smooth typography, and tailored micro-animations.
- **Responsive & Performant**: Built with modern web technologies for optimal performance across devices.
